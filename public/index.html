03 / VOICE

Paste the script — narration is generated on this machine, free, as many takes as you want. Engine: Voicebox (MIT — start the app first). Fallbacks: piper / espeak-ng / say for scratch tracks.

04 / ASSEMBLE

Stitch visual + voiceover into one MP4 with word-by-word captions timed to the beat (Whisper word timestamps). Output is a CRF-18 mezzanine — delivery encode happens in EXPORT.

05 / SCORE

The improvement loop: render → score → fix the flagged seconds → re-score, until the curve holds. Built-in curve is a local attention proxy (audio energy + cut density + speech density). For brain-response scoring, install TRIBE v2 yourself — non-commercial license, never bundled.

TRIBE V2 — BRAIN MODEL (SELF-INSTALL)

06 / EXPORT
